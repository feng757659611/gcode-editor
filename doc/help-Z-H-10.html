Chapter 8

快捷键

Ctrl + N 新建文件
Ctrl + S 保存文件
Ctrl + O 打开已有文件
Ctrl + Z 撤消
Ctrl + Y 重做
Ctrl + X 剪切
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + A 全选
Ctrl + F 查找
Ctrl + H 替换
Alt + M 快速注释反注释
Alt + N 自动补全
F5 编译
Ctrl + F5 编译并运行
F6 重排版
F7 函数列表
F8 生成函数注释
F9 状态栏
Tab 进行Tab操作时，TAB字符转换成4个空格宽度
