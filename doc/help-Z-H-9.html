Chapter 7

工具

7.1  编译

7.1.1  编译器设置

  • 功能说明

    设置编译器的具体路径、临时文件位置、编译器的参数

  • 使用方法

    • 编译器
      Windows 平台上请选择具体 exe 文件；在 Linux 平台上请输入编译器的绝对路径。

    • 临时文件位置
      编译出的可以执行文件存放位置。

    • 编译参数
      如果用到相应的库文件，请手动添加相应的编译参数。

  • 注意事项

    • cPad 默认使用的编译器为 tcc

    • 临时文件位置在当前编辑文件的目录

    • 编译参数默认为 -Wall

    • Linux 平台下中文路径可能会导致编译失败

7.1.2  编译

  • 功能说明
    编译当前文件

  • 使用方法
    点击即编译当前文件。编译信息输出到编译信息的窗口。可点击“清除”来清除编译信息窗口的内容。

  • 注意事项
    cPad 的编译设置默认使用 -o 参数。如果当前文件为 test.c，则编译选项默认为：<code>编译器 -o 临时文件位置+test.exe 当前位置+test.c</code>

7.1.3  编译并运行

  • 功能说明
    编译并运行当前文件

  • 使用方法
    点击即做编译和运行操作。编译信息输出到编译信息的窗口。可点击“清除”来清除编译信息窗口的内容。

  • 注意事项
    运行时，Windows 平台弹出 cmd 并运行当前程序；Linux 平台直接运行编译出的可执行文件。Linux 用户可能查看不了程序输出，解决办法是打开终端直接运行。

7.2  重排版

  • 功能说明

    • cPad支持对C语言代码的“重排版”功能。

    • "重排版"操作是针对编辑器中的C语言代码，进行重新排版，使代码格式更规范、具有可读性。

  • 使用方法

    • 用户点击菜单栏的“工具”选项，选择“重排版”；或者点击工具栏中的“重排版”按钮，可进行“重排版”操作。

    • 点选后，会自动弹出“重拍版”窗口，其中有6种选项可选，用户可以按照自己的编程风格，进行组合选择。

    • 强制在声明后断行：选择此项后，重排版的代码，会在所有的函数声明、变量声明、常量声明后添加空行（函数声明与全局变量声明之间不添加）；若未选，则不会添加此空行。

    • 强制在函数体后断行：选择此项后，重排版的代码，会在所有的函数体后添加空行（最后一个函数后如果没有其他代码，则不会添加）；若未选，则不会添加此空行。

    • 左大括号单独一行：选择此项后，重排版的代码，会使所有语句块开端的左大括号单独占用一行；若未选，该左大括号将会位于语句块关键词行的末尾。

    • 分号、逗号前加空格：选择此项后，重排版的代码，会使所有分号、逗号前添加空格；若未选，则不会添加该处空格。

    • 括号内侧加空格：选择此项后，重排版的代码，会在所有小括号、中括号内侧（左括号的右边、右括号的左边）添加空格（空括号只会添加一个空格；若已经存在其他空格，则不会再次添加）；若未选，则不会添加该处空格。

    • 函数名、区块关键词后加空格：选择此项后，重拍版的代码，会在所有函数名、区块关键词（包括if、else if、for、do、while、switch）后添加空格；若未选，则不会添加该处空格。

  • 注意事项
    如果重排版前，代码存在语法错误，重排版可能导致一定程度上的代码混乱，建议先使用编译功能，清除代码错误后再进行重排版。

7.3  函数列表

  • 功能说明

    • cPad支持列出C语言代码的"函数列表"功能。

    • "函数列表"操作是根据编辑器中的C语言代码，提取已经或正在编写的函数信息，列出函数列表，供用户查看、跳转。

  • 使用方法

    • 用户点击菜单栏的“工具”选项，选择“函数列表”；或者点击工具栏中的“函数列表”按钮，可弹出函数列表。

    • 弹出的函数列表在主编辑栏上方、工具栏下方，默认写有“<Please Select>”文字。</Please>

    • 点击该文字所在下拉菜单，将会跳出当前编辑器中C语言代码所有已经或正在编写的函数信息（包括函数类型、函数名、函数的参数表）。

    • 当用户点击列表中某一函数信息时，主编辑栏会自动跳转至该函数区域。

  • 注意事项

    • 用户对某函数只进行了声明，并没有进行编写的，不会出现在函数列表中。

    • 用户对某函数的编写，只要已经写出了函数的类型、名称、参数表以及左大括号，便认为该函数处于正在编写状态。

    • 用户每次点击下拉菜单时，函数列表会根据当前代码进行刷新，未进行点击时，列表不会刷新。

7.4  快速注释与反注释

  • 功能说明
    可对光标所在行的输入内容进行快速注释；可对已有注释语句取消注释

  • 使用方法
    在需要注释或者取消注释的的行按组合键 Alt + M，或者点击工具栏中的快速注释按钮

  • 注意事项
    该功能只可实现单行的快速注释与反注释

7.5  生成函数注释

  • 功能说明
    在函数头生成当前函数的标准注释

  • 使用方法
    将光标移动到函数实体开始，按快捷键F7或者点击工具栏的生成函数注释按钮

  • 注意事项
    光标必须要在函数实体的开始才能生成函数的注释

7.6  自动补全

  • 功能说明

    • cPad支持列出C语言词语的“自动补全”功能。

    • "自动补全"操作是根据编辑器中的C语言代码，提取已经存在的词语信息，对用户当前正在编辑的词语进行补全。

  • 使用方法

    • 用户点击菜单栏的“工具”选项，选择“自动补全”；或者点击工具栏中的“自动补全”按钮，可弹出补全列表。

    • 弹出的补全表包含所有当前代码中已经出现过的、以当前光标位置前的半词开头的词语。当词语数量超过5个时，会出现上下方向的拉条。

    • 在列表中，词语以字典序排列，按键盘的上下键可以进行选择，在正确的词语上点击“回车”键即可完成补全。

    • 列表弹出后，可以继续进行编辑，以减少可补全选项。

    • 当用户输入空格或者其他使该词不能补全的键后，补全框会自动消失。

    • 当用户希望取消补全时，需要按键盘的“ESC”键。

  • 注意事项
    当补全框跳出后，鼠标点击暂时失效。当补全完成或者用户取消补全后，鼠标点击恢复。
