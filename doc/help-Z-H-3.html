cPad 简介

cPad 是 5-Up 开发小组用 Qt 开发的一款 C 语言编辑器。在 C 语言文件处理上具有很多良好的特性，比如 C 语言语法高亮、进行 C 文件重排版、显示函数列表、自动补全、快速注释当前行、生成函数的标准注释、编译运行文件……除此之外，cPad还有一些个性化的设置，比如用户可以选择颜色主题，可以开关当前行高亮和语法高亮……由于 cPad 是一门叫“软件基础实习”课的产物，完成时间在一个半月左右，所以软件难免会有问题。因为课程的性质，决定了cPad只能是一个试验品，不会再有大的改进，但我们愿意和您交流和 cPad 有关的所有东西！
